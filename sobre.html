Violet é uma marca fictícia de roupas e streetwear voltada especificamente para o público skatista. O projeto foi idealizado e desenvolvido durante as aulas da disciplina de Interface Web, sob a orientação do professor Bruno, como parte de uma avaliação bimestral. A proposta do trabalho exigia a criação de uma plataforma digital com tema livre. A escolha pelo universo do skate permitiu explorar uma identidade visual urbana, moderna e cheia de atitude. O layout do site foi planejado para refletir o dinamismo do esporte, priorizando a usabilidade, o apelo visual das peças de vestuário e uma navegação fluida e intuitiva para o usuário.
